# Eatsee Foods - Modern Food Website

A premium, production-ready website for **Eatsee Foods**, a traditional food manufacturing and delivery company based in Omassery, Kerala.

## Tech Stack
- **React.js** (Vite)
- **GSAP & ScrollTrigger** - Storytelling scroll animations
- **Tailwind CSS** - Modern, responsive styling
- **Framer Motion** - UI micro-animations
- **React Router** - Multi-page navigation
- **React Context** - Dark mode management
- **Lenis** - Smooth scrolling experience

## Key Features
- **Storytelling Scroll Showcase**: A unique GSAP-powered section highlighting signature food products (Pathiri, Idiyappam, Vellappam, Chappathi).
- **Dark Mode**: Fully implemented dark/light mode toggle with persistence.
- **Responsive Design**: Optimized for all devices from mobile to ultra-wide screens.
- **WhatsApp Integration**: Floating button for direct customer inquiries.
- **Comprehensive Pages**: Home, About, Products, Services, Gallery, Contact, and Order Request.
- **Premium Aesthetics**: High-end typography, glassmorphism, and smooth transitions.

## Project Structure
- `/src/components`: Reusable UI elements (Navbar, Footer, Hero, etc.).
- `/src/pages`: Individual page components and layouts.
- `/src/context`: Global state management for theming.
- `/public/image`: Brand assets and food photography.

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
